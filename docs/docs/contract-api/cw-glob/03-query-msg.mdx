# `QueryMsg`

The `QueryMsg` is the message that allows you to see the state of the contract. Therefore, it is important to
understand the state of the contract.

## `GlobHash`

```rust reference
https://github.com/srdtrk/cw-ica-controller/blob/v0.4.0/src/types/msg.rs#L103-L105
```

This message is used to query the sha256sum of a stored wasm blob.
