# `QueryMsg`

The `QueryMsg` is the message that allows you to see the state of the contract. Therefore, it is important to
understand the state of the contract.

## `GlobHash`

Retrieves the sha256sum hash of the stored globs.

HashGlob  | Description | Type |
--- | --- | --- | 
`keys` | array of storage keys associated to blob stored internally  | Array of String
